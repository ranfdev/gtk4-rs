Converts an accelerator keyval and modifier mask into a string parseable by gtk_accelerator_parse_with_keycode().

This is similar to accelerator_name() but handling keycodes. This is only useful for system-level components, applications should use accelerator_name() instead.

display

a gdk::Display or None to use the default display

accelerator_key

accelerator keyval

keycode

accelerator keycode

accelerator_mods

accelerator modifier mask

Returns

a newly allocated accelerator name.
